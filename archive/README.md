# Legacy Vim snapshot (`legacy-dot-vimrc`)

Neovim is the maintained surface of this repo. This file was the original **`~/.vimrc`** when the repository was bootstrapped; it stays here so nothing is rewritten out of Git history.

- **Active config:** [`nvim/`](../nvim/) — copy or symlink those files into `~/.config/nvim/`.
- **Recover old versions:** `git log -p archive/legacy-dot-vimrc` (or checkout an older commit).

No future changes are planned here unless you consciously revive classic Vim parity.
